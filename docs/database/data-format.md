# Формат данных для наполнения базы данных

## Обзор

Эта документация описывает правильный формат данных для добавления слов в базу данных приложения изучения русского языка.

## Структура модели Word

### Поля базы данных

| Поле | Тип | Обязательное | Описание |
|------|-----|-------------|----------|
| `word` | string(128) | ✅ | Слово с пропущенной частью |
| `missing_part_index` | integer | ✅ | Индекс позиции пропущенной части |
| `explanation` | string(2048) | ❌ | Объяснение написания правильного ответа |
| `answers` | JSON array | ✅ | Массив вариантов ответов |
| `right_answer` | string(128) | ✅ | Правильный ответ |
| `task_number` | integer | ❌ | Номер задания ЕГЭ |
| `category_id` | integer | ✅ | ID категории (Foreign Key) |

## Правила оформления

### 1. Поле `word` (слово с пропусками)

**Формат:** Пропущенная часть игнорируется.

**Примеры:**
- ✅ `"првосходный"` - пропущена буква 'е'
- ✅ `"бепредельный"` - пропущена буква 'з' 
- ✅ `"дача"` - пропущена приставка 'от'

### 2. Поле `missing_part_index`

**Формат:** Индекс начала пропущенной части (начинается с 0).

**Примеры:**
- `"првосходный"` → `missing_part_index: 2`
- `"дача"` → `missing_part_index: 0`
- `"бепредельный"` → `missing_part_index: 2`

### 3. Поле `answers` (варианты ответов)

**Формат:** JSON массив строк. 

**Структура:**
```json
[
  "правильный_ответ",
  "неправильный_ответ_1", 
  "неправильный_ответ_2",
  "неправильный_ответ_3"
]
```

**Примеры:**
```json
["е", "и", "я", "о"]
["з", "с", "ц", "т"]  
["от", "под", "над", "в"]
```

### 4. Поле `right_answer`

**Формат:** Строка с правильным ответом (один из элементов `answers`).

**Примеры:**
- Для `"првосходный"` → `"е"`
- Для `"бепредельный"` → `"з"`
- Для `"дача"` → `"от"`

### 5. Поле `explanation` (объяснение)

**Формат:** Краткое объяснение написания правильного ответа.

**Примеры:**
- `Превосходный - пре- в значении 'очень'`
- `Беспредельный - 'с' перед глухой согласной`

## Примеры правильного оформления

### Пример 1: Приставки
```json
{
  "word": "првосходный",
  "missing_part_index": 2,
  "explanation": "пре- в значении 'очень'",
  "answers": ["е", "и", "я", "о"],
  "right_answer": "е",
  "category_id": 1
}
```

### Пример 2: Корни
```json
{
  "word": "бепредельный", 
  "missing_part_index": 2,
  "explanation": "'с' перед глухой согласной",
  "answers": ["з", "с", "ц", "т"],
  "right_answer": "з",
  "category_id": 2
}
```

## Категории

Перед добавлением слов убедитесь, что категория существует в таблице `category`:

| ID | Название | Описание |
|----|----------|----------|
| 1 | Приставки | Задания на правописание приставок |
| 2 | Корни | Задания на правописание корней |
| 3 | Суффиксы | Задания на правописание суффиксов |
| 4 | Окончания | Задания на правописание окончаний |

### Рекомендации:

- Используйте минимум вариантов ответа (2-4)
- Делайте неправильные ответы правдоподобными
- Пишите краткие, понятные объяснения
- Группируйте слова по категориям
